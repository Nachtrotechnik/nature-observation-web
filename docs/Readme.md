Lernziel

In dieser Programmieraufgabe geht es um den grundlegenden Aufbau von Webseiten und
die grundlegenden Programmierkonzepte im WWW (insb. Wiederholung von HTML und
CSS). Das Aussehen der Anwendung wird nicht in die Bewertung mit einbezogen. Ein schönes Design ist aber gern gesehen.

Aufgaben

In dieser Aufgabe sollen Sie die Grundstruktur für eine „Onlineplattform zum Erfassen und
Anzeigen von naturbezogenen Beobachtungen“ für Tiere und Pflanzen aufbauen. In diesem Portal sollen Nutzer*innen unter anderem:

    • eine Liste mit Beobachtungen auf einer Karte und als Kacheln sehen können
    • eine Beobachtung auswählen können, um Detail-Informationen einzublenden
    • eine Beobachtung auswählen können, um sie auf der Karte hervorzuheben
    • die Liste der Beobachtungen nach bestimmten Kriterien filtern bzw. darin nach bestimmten Beobachtungen suchen können
    • interessante Beobachtungen selektieren können
    o Häkchen setzen bzw. zur Wunschliste hinzufügen
    o Liste mit selbst ausgewählten Beobachtungen anzeigen lassen
    • eigene Beobachtung hinzufügen können

Das „Natur-Beobachtungs-Portal“ soll im Laufe des Semesters als Web-Anwendung umgesetzt werden. In dieser Aufgabe sollen Sie hierfür zunächst ein Mockup für die Web-Anwendung erstellen. Dieses wird dann von Ihnen in den kommenden Aufgaben durch Funktionen erweitert werden.

1. Layout der Webseite mit HTML

Entwerfen Sie mithilfe von HTML ein erstes Layout der Webseite für das „Natur-Beobachtungs-Portal“. Die Webseite kann hierfür in Unterseiten unterteilt werden. Sie sind frei in der Gestaltung, die folgenden Seiten könnten jedoch beispielsweise vorhanden sein:

    • Hauptseite mit Karte und Liste von Beobachtungen
    • einzelne Beobachtung mit Details
    • Login und Registrierung
    • Kontakt und Impressum

Zudem sollten Sie überlegen, welchen Grundaufbau die Seiten haben, beispielsweise:

    • Header mit Logo und Namen der Webseite
    • Links zum Registrieren und Einloggen
    • Karte und Liste der Beobachtungen als Kacheln
    • Such- und Filterfunktion
    • Footer mit Links zum Kontakt und Impressum

Die folgenden Kriterien müssen hierbei erfüllt werden:

    • Die wichtigsten Meta-Tags (mindestens zwei) und der Titel sind gesetzt.
    • Es ist mindestens eine externe CSS-Datei korrekt eingebunden.
    • Es sind mindestens 30 unterschiedliche Elemente enthalten, darunter semantische
    Container, Überschriften, Tabellen, Formulare und Links.
    • Von der Hauptseite aus ist über Links mindestens eine Unterseite für die Registrierung erreichbar.

Hierbei können Sie sich von bereits existierenden Webseiten für Naturbeobachtungen, wie zum Beispiel NABU naturgucker, inspirieren lassen.

2. Darstellung der Webseite mit CSS

Gestalten Sie mithilfe von CSS das Aussehen der Webseite. Legen Sie hierfür eine Datei an, die Sie in Ihre Webseite einbinden können.
Die folgenden Kriterien müssen hierbei erfüllt werden:

    • Es sind mindestens zehn Regeln mit Elementen und Klassen und vier anderen Se-
    lektoren enthalten.
    • Es sind mindestens 30 verschiedene Eigenschaften enthalten.
    • Die Regeln für ein Responsive Web Design (RWD) sind eingehalten.
    Das Ergebnis sollte eine optisch ansprechende Website sein.

Abnahme

Das Ergebnis der Aufgaben ist in Moodle hochzuladen. Komprimieren Sie hierfür alle notwendigen Dateien (HTML, CSS, JavaScript, usw.) in ein zip-Archiv und laden Sie dieses im entsprechenden Abgabebereich hoch. Sie können sich bei der Bearbeitung der Aufgabe gegenseitig unterstützen, eine Gruppenarbeit (als Abgabe) ist allerdings nicht zulässig.
