# MongoDB Collection Schema: Observations

## Collection Name
`observations`

## Database Schema

### Observation Document Structure

```javascript
{
  _id: ObjectId,           // Auto-generated by MongoDB
  title: String,           // Required - Title of the observation
  latinName: String,       // Required - Latin/scientific name
  location: String,        // Required - Location/place where observed
  date: Date,             // Required - Date of observation
  createdAt: Date,        // Auto-generated timestamp
  updatedAt: Date         // Auto-generated timestamp
}
```

## Mongoose Schema Definition

```javascript
const mongoose = require('mongoose');

const observationSchema = new mongoose.Schema({
  title: {
    type: String,
    required: true,
    trim: true
  },
  latinName: {
    type: String,
    required: true,
    trim: true
  },
  location: {
    type: String,
    required: true,
    trim: true
  },
  date: {
    type: Date,
    required: true
  }
}, {
  timestamps: true  // Automatically adds createdAt and updatedAt
});

const Observation = mongoose.model('Observation', observationSchema);
module.exports = Observation;
```

## Field Descriptions

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `_id` | ObjectId | Auto | Unique identifier automatically generated by MongoDB |
| `title` | String | Yes | Title/name of the observation (e.g., "Red Fox Sighting") |
| `latinName` | String | Yes | Scientific/Latin name of the species (e.g., "Vulpes vulpes") |
| `location` | String | Yes | Location where the observation was made (e.g., "Berlin, Germany") |
| `date` | Date | Yes | Date when the observation was made |
| `createdAt` | Date | Auto | Timestamp when the document was created (added by timestamps option) |
| `updatedAt` | Date | Auto | Timestamp when the document was last updated (added by timestamps option) |

## Indexes (Recommended)

```javascript
// Create index on date for efficient date-based queries
observationSchema.index({ date: -1 });

// Create index on latinName for species-based searches
observationSchema.index({ latinName: 1 });
```

## Validation Rules

- All fields (title, latinName, location, date) are required
- String fields are automatically trimmed of whitespace
- Date field must be a valid date
- Timestamps are automatically managed by Mongoose
